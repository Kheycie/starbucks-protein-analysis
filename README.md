# Starbucks Protein Analysis

## Project Overview
This project analyzes protein content across Starbucks menu items, including drinks, bakery items, sandwiches, and parfaits. The goal is to:

- Identify the top protein items
- Explore protein distribution across item types
- Highlight items with the highest protein efficiency (protein per calorie)

This project is designed as a beginner-friendly portfolio piece for data analysis.

## Dataset
The dataset (`starbucks_data.csv`) includes:

- `item`: Name of the Starbucks menu item
- `calories`: Calories per serving
- `fat`: Fat content (g)
- `carb`: Carbohydrates (g)
